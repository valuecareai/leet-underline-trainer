다 그었으면 아래 "채점하기" 버튼을 눌러주세요.
지 문
공리주의는 행위의 도덕적 옳고 그름이 그 행위가 산출하는 쾌락과 고통의 총량에 의해 결정된다고 주장한다. 그러나 이 입장은 분배의 문제를 간과한다는 비판을 받는다. 예컨대, 소수의 극심한 고통을 감수하더라도 다수의 작은 쾌락의 합이 더 크다면, 공리주의는 그 행위를 정당화할 수 있기 때문이다. 이에 대해 롤스는 정의의 원칙이 공리의 극대화보다 우선해야 한다고 반론한다. 그는 합리적 인간이 '무지의 베일' 뒤에서 선택한다면, 최소 수혜자의 이익을 최대화하는 원칙, 즉 '맥시민 원칙'을 선택할 것이라고 주장한다.
0%
핵심 포인트 일치율

❓ 위 글의 내용과 일치하는 것은?

①
공리주의는 분배의 문제를 핵심 원리로 삼는다.
②
롤스는 공리주의의 목적 자체를 부정한다.
③
공리주의에 따르면 다수의 이익이 소수의 고통을 정당화할 수 있다.
④
맥시민 원칙은 다수의 쾌락 총량을 극대화하는 것을 목표로 한다.
⑤
무지의 베일은 공리주의의 개념이다.

🧠 판단 과정

① → X : "간과한다"고 했으므로, 핵심 원리로 삼는다는 것은 반대
② → X : 롤스는 '우선순위'를 말했지, 공리주의 목적 자체를 부정하진 않음
③ → ✅ 정답 : "소수의 고통 감수 + 다수의 쾌락 합이 크면 → 정당화 가능" 과 일치
④ → X : 맥시민은 최소 수혜자 이익 최대화 (다수 쾌락 극대화 ≠)
⑤ → X : 무지의 베일은 롤스의 개념

Part 2 · 추리논증 — 유형 1

드래그로 밑줄을 그어보세요!
기호화가 필요한 조건문, 부정 조건, 확정 사실을 찾아 드래그하세요.
다 그었으면 아래 "채점하기" 버튼을 눌러주세요.
조 건
A가 참석하면 B도 참석한다.
C가 참석하지 않으면 D가 참석한다.
B와 D는 동시에 참석하지 않는다.
A는 참석한다.
A 참석 (확정)
→ [조건 1] A → B   ∴ B 참석 (확정)
→ [조건 3] B이면 ¬D   ∴ D 불참 (확정)
→ [조건 2] ¬C → D 의 대우: ¬D → C   ∴ C 참석 (확정)
0%
핵심 포인트 일치율

❓ 반드시 참인 것은?

①
B는 참석하지 않는다.
②
C는 참석한다.
③
D는 참석하지 않는다.
④
C와 D 모두 참석한다.
⑤
B는 참석하고 C는 참석하지 않는다.

🧠 판단 과정

① B 불참 → X (B는 참석 확정)
② C 참석 → ✅ 정답 (대우를 통해 C 참석 확정)
③ D 불참 → 참이지만, ②가 더 직접적인 논리 도출
④ C+D 모두 참석 → X (D는 불참 확정)
⑤ B참석+C불참 → X (C 참석 확정)

🔑 대우 공식 (반드시 암기!)

원래 명제: A → B   ✅ 참
대우: ¬B → ¬A   ✅ 반드시 참
역: B → A   ❌ 참 아닐 수 있음
이: ¬A → ¬B   ❌ 참 아닐 수 있음
💡

줄 치는 포인트

조건문에서 "~이면", "~할 때", "~인 경우"가 보이면 즉시 기호화하세요. 기호화가 곧 풀이의 시작입니다!

Part 2 · 추리논증 — 유형 2

전제 (근거)

드래그로 밑줄을 그어보세요!
전제(근거), 인과관계, 결론을 구분하며 드래그하세요.
다 그었으면 아래 "채점하기" 버튼을 눌러주세요.
주 장
규칙적인 운동은 우울증을 감소시킨다. 연구에 따르면 주 3회 이상 운동한 집단은 그렇지 않은 집단보다 우울증 수치가 유의미하게 낮았다.
0%
핵심 포인트 일치율

❓ 위 논증을 약화하는 것은?

①
운동을 많이 한 집단은 사회경제적 지위도 높았다.
②
우울증 환자 중 일부는 운동 능력이 저하된다.
③
운동의 효과는 개인차가 있다.
④
연구에 참여한 집단은 운동 외에도 식단을 개선했다.
⑤
우울증 감소에는 수면도 중요하다.

🧠 판단 과정 — 약화 = 인과관계를 끊는 것

① 사회경제적 지위 → 제3변수 가능 → 약화 가능 (부분 약화)
② 우울증→운동 저하 → 역인과, 논증과 다른 방향
③ 개인차 → 일반적 약화이나 결정적이지 않음
④ 식단 개선이 원인일 수 있음 → ✅ 정답 교란변수 (실험 통제 실패)
⑤ 수면도 중요 → 운동의 역할을 직접 부정하지 않음
💡

약화 문제 공식

약화 = 인과관계를 끊는 것! 교란변수(제3의 원인), 역인과(원인과 결과 뒤바뀜), 통제 실패(다른 변수 미통제)를 찾으세요.
